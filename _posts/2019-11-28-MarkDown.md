## Header 2
### Header 3
```

## 2. 강조
```
이탤릭체는 *별(asterisks)* 또는 _언더바(underscore)_
볼드체는 **별(asterisks)** 2개 또는 __언더바(underscore)__ 2개
취소선은 ~~물결~~ 2개
